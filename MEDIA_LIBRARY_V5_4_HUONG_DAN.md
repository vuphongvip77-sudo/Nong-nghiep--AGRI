# AGRI Media Library v5.4

## Đường dẫn
`/platform/media/`

## Chức năng
- Quản lý hình ảnh
- Quản lý video
- Quản lý PDF, Word và Excel
- Tìm kiếm, lọc và sắp xếp
- Thêm, sửa và xóa thông tin tệp
- Nhập và xuất `media.json`

## Lưu ý
GitHub Pages không thể nhận tệp tải trực tiếp từ trình duyệt vào repository. Media Library v5.4 quản lý danh mục và đường dẫn tệp. Các tệp thật vẫn cần được đưa vào thư mục `media/` trên GitHub khi xuất bản.
