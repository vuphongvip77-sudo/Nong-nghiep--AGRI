# Hướng dẫn AGRI Studio v1

## Mở Studio
Truy cập đường dẫn:

`https://vuphongvip77-sudo.github.io/Nong-nghiep--AGRI/studio/`

## Cách thêm bài
1. Chọn **Thêm bài viết**.
2. Nhập tiêu đề, chuyên mục, mô tả và từ khóa.
3. Thêm các phần nội dung.
4. Bấm **Lưu bài viết**.

Dữ liệu được lưu tạm trên chính trình duyệt đang sử dụng.

## Cách đưa bài lên website
1. Trong AGRI Studio, bấm **Xuất articles.json**.
2. Vào repository GitHub.
3. Mở thư mục `data`.
4. Thay file `articles.json` cũ bằng file vừa xuất.
5. Commit thay đổi.
6. Chờ GitHub Pages cập nhật.

## Lưu ý quan trọng
GitHub Pages là website tĩnh nên AGRI Studio không thể tự ghi trực tiếp vào repository nếu chưa có máy chủ hoặc GitHub API có xác thực. Bản v1 giúp soạn, quản lý và xuất dữ liệu mà không phải sửa mã JSON bằng tay.
