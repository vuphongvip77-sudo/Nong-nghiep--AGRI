# AGRI Publish Engine v5.3

## Đường dẫn
`/platform/publish/`

## Quy trình
1. Biên soạn trong AGRI Editor.
2. Lưu dữ liệu.
3. Mở Publish Engine.
4. Bấm **Kiểm tra dữ liệu**.
5. Sửa các lỗi nếu có.
6. Bấm **Xuất bản**.
7. Bấm **Tải toàn bộ**.
8. Đưa các file vào đúng thư mục trên GitHub.

## Kiểm tra tự động
- Tên đối tượng
- Mã đường dẫn
- Loại tri thức
- Chuyên mục
- Biểu tượng
- Mô tả
- Các mục nội dung
