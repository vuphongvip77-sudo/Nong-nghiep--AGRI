Đưa đối tượng vào Knowledge Base

Chọn file đối tượng đã xuất từ AGRI Studio, kiểm tra thông tin rồi tạo hai file sẵn sàng đưa lên GitHub.

Chưa chọn file.

Sau khi tải file

  1. Đưa file đối tượng vào đúng thư mục knowledge/....
  2. Đưa knowledge.json vào knowledge/index/.
  3. Commit changes trên GitHub.
