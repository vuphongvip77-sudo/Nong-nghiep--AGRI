# AGRI Website v1

Bộ mã nguồn tĩnh dành cho GitHub Pages.

## Cài đặt
1. Sao lưu repository hiện tại.
2. Chép toàn bộ nội dung trong thư mục này vào thư mục gốc của repository.
3. Commit và push.
4. Mở GitHub Pages và làm mới bằng `?v=3`.

## Cấu trúc
- `index.html`
- `pages/`
- `assets/css/`
- `assets/js/`
- `data/`
- `docs/`

Không cần npm hoặc máy chủ. Website dùng HTML, CSS và JavaScript thuần.

## Thêm bài viết
Mở `data/articles.json`, sao chép một phần tử mẫu và sửa nội dung.
